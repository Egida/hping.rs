## Build on Debian and Ubuntu

```sh
 sudo apt install cargo

 git clone https://github.com/lidarbtc/hping.rs.git
 cd hping.rs

 cargo build --release
```

## Usage

```sh
 Use command line : ./hping-rs <ip:port>
      └──────────> ./hping-rs 127.0.0.1:80
```
